# Project: Alie
Ce projet crée un assistant inintelligent capable 
d'interagir avec l'utilisateur via des commandes vocales ou textuelles à coup de regex !



# Applications

- [Alie API](https://github.com/etneuhct/alie_apix)

Serveur Django interagit avec la BDD pour le stockage de données à long terme.
- Alie Client

Client Angular permettant d'envoyant des commandes vocales, ..., à "Alie"

- [Alie Brain](https://github.com/etneuhct/alie_brain)

Serveur Fastapi contenant la logique et les informations à court terme de l'assistant
